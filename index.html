The Project

If Captain Obvious stated to you over a cocktail that there is a definite relationship between latitude and observed temperature for a given location, you would probably agree without hesitation. But how would one prove it?

Summary

In this project, on July 1, 2019, I used the Python numpy library to randomly generate the latitude and longitude of 1,500 locations. I then used the Python citypy libary to find the nearest city to each location. Next, I used the Python openweathermapy wrapper to access the openweatthermap.org api and get then-current weather data for each city. After discarding duplicate cities and cities for which there was no current weather data, I ended up with 568 data points.

I then used the Python matplotlib library to create a scatterplot for each of four different current weather attributes available in the data:

  • maximum temperature
  • relative humidity
  • cloudiness
  • wind speed

The data showed a pronounced relationship between latitude and temperature, but—at least for this one day in July—there did not seem to be a strong relationship between latitude and the three other weather metrics that I considered.
