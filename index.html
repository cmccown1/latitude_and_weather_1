The Project

If your friend, Captain Obvious, stated to you over dinner that there is a definite relationship between latitude and observed temperature for a given location, you would probably agree without hesitation. But how would you prove it?

Summary

To begin this project, I randomly generated the latitude and longitude of 1,500 geographic coordinates using the Python numpy libary. Next, using the Python citypy libary, I found the nearest city to each geographic coordinate. Finally, using the Python openweathermapy wrapper, I retrieved then-current weather data—it was July 1, 2019—via the openweatthermap.org api. After discarding duplicate cities and cities for which there was no current weather data, I ended up with 568 data points.

Next, I created a scatterplot, using the Python matplotlib library, for each of four different weather attributes that I chose from the data:

  • maximum temperature
  • relative humidity
  • cloudiness
  • wind speed

The data showed the expected pronounced relationship between latitude and temperature, but—at least for this one day in July—there did not seem to be a strong relationship between latitude and the three other weather metrics that I considered.
