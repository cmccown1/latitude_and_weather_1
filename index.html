The Project

Summary

In this project, I used the Python numpy library to randomly generate the latitude and longitude of 1,500 locations. I then used the Python citypy libary to find the nearest city to each location. At this point, I used the Python openweathermapy wrapper to access the openweatthermap.org api and get then-current weather data for each city. After discarding duplicate cities and cities for which there was no current weather data, I ended up with 568 data points.

I then used the Python matplotlib library to create a scatterplot for each of four different current weather attributes available in the data:

  • maximum temperature
  • relative humidity
  • cloudiness
  • wind speed
